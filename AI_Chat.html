🤖 企业私有 AI Chatbot（基于 DeepSeek）

构建于本地的企业级智能问答系统，融合 RAG+LLM 技术，支持知识库自学习，保护数据隐私。

🔧 技术栈

Python Flask FAISS DeepSeek HTML/CSS JavaScript JSON

📌 项目背景

由于企业员工频繁使用公共聊天平台，存在较高的数据泄露风险。因此我们构建一个本地运行的 AI Chatbot，专为企业内部服务。

🧠 实现方式

  • 前端使用 HTML、CSS、JavaScript 构建用户界面
  • 后端基于 Flask 构建 API 服务
  • 采用 RAG + FAISS 实现知识库问答能力
  • 调用 LLM（DeepSeek）进行智能分析与回答
  • 用户问题自动保存至本地 JSON 数据库，持续增强知识库
  • 项目部署于本地 4090 GPU 服务器，确保数据不出企业内网

📊 项目成果

  • 问答准确率高达 85%
  • 已在部分部门试运行，员工反馈良好
  • 下一阶段将推广至公司全部部门
